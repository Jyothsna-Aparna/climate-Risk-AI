# 🌍 Global Climate Risk & Early-Warning AI Radar

An interactive, real-time climate risk dashboard supporting **UN SDG 13 (Climate Action)**. Built using Python, Streamlit, and geospatial analytics.

🚀 **[Click Here to Open Live Demo App](https://climate-risk-ai-yantuu8lwnf8rambs6kqj6.streamlit.app)**

---

## 📸 Overview
This platform bridges live weather data APIs with geospatial visualization to deliver real-time risk assessments for Floods, Extreme Heat, and Wildfires worldwide.

## ✨ Key Features
- **Real-Time Data Integration:** Connects with Open-Meteo APIs for dynamic rainfall, wind, and temperature analytics.
- **AI Threat Assessment:** Algorithmic calculation of threat scores and localized early-warning indicators.
- **Interactive Geospatial Mapping:** Color-coded heatmaps and warning overlays rendered via Folium.
- **Multilingual Support:** Native interface in English, Telugu, and Hindi.

## 🛠️ Tech Stack
- **Frontend / UI:** Streamlit (Custom Glassmorphism CSS)
- **Data Analytics:** Python, Pandas
- **Geospatial & Visuals:** Folium, Streamlit-Folium
- **APIs:** Open-Meteo Weather & Geocoding API
